# Sistema para Seleção Simplificada de professores (Demanda DEAD/IFMA)

Sistema Feito pelo grupo 2 para a obtenção de nota na disciplina de Analise de Sistemas 2

Para clonar do main:  
`` git clone https://github.com/MaylaNascimento/SIS_PROF_EAD.git`` 
  
Para clonar do dev:  
`` git clone -b dev https://github.com/MaylaNascimento/SIS_PROF_EAD.git`` 
  
Para iniciar em modo desenvolvimento:  
``npm run dev `` 

Para iniciar em modo real, sem logs:  
`` npm run start ``
